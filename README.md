# tslab

Typescript lab with pixijs

# install dependencies

```
npm install pixi.js
npm install -D vite typescript
```

# setting the package.json

```
  "scripts": {
    "dev":"vite",
    "bulid":"tsc && vite build",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
```

# init ts
```
npx tsc --init
```
